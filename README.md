## 安装教程

### 环境要求

- Node.js 14.0 或更高版本
- - npm 6.0 或更高版本
 
  - ### 安装步骤
 
  - 1. 克隆项目到本地
   
    2. ```bash
       git clone https://github.com/fengxinyu96325/My-project.git
       ```

       2. 进入项目目录
      
       3. ```bash
          cd My-project
          ```

          3. 安装依赖
         
          4. ```bash
             npm install
             ```

             4. 运行项目
            
             5. ```bash
                npm start
                ```

                ### 配置说明
